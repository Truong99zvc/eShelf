# eShelf MLOps Skeleton

Thư mục này chứa skeleton cho phần MLOps/AIops của đồ án:

- `training/`: script train model gợi ý sách (Python).
- `serving/`: FastAPI model server (tuỳ chọn) – hiện tại dùng Node ML service mock.
- `mlflow/`: config để chạy MLflow tracking server (tuỳ chọn).

Các file này đủ để mô tả kiến trúc MLOps trong báo cáo, bạn có thể mở rộng tuỳ ý.
